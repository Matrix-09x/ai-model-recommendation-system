# AI Model Recommendation System

A NumPy-based AI model recommendation system that compares and analyzes 
AI models using cosine similarity and vector mathematics.

## What It Does
Given an AI model, the system recommends the most similar models 
based on their capability vectors — the same core concept used in 
Netflix recommendations, semantic search, and RAG systems.

## Features
1. Overall performance analysis — mean, ranking
2. Strongest and weakest capability per model
3. Consistency analysis — balanced vs specialized models
4. Best model per metric — top coding, reasoning, speed model
5. Similarity engine — cosine similarity between any two models
6. Recommendation system — find most similar model
7. Most and least similar pairs
8. Similarity matrix — every model vs every model
9. Cost vs Performance categorization

## Core Concept Learned
Meaning → numbers → vectors → cosine similarity → recommendations
This is the same math powering embeddings, semantic search, 
vector databases, and RAG systems.
